# Semana 1: Fundamentos y Big Data

## Resumen

Durante la Semana 1 se exploraron los fundamentos de la Ciencia de Datos y Big Data, abordando tanto bases matemáticas y algorítmicas como aplicaciones reales en industrias como salud y finanzas. La semana culminó con un caso práctico integral sobre DeportivaMX, una tienda deportiva en línea, donde se diseñó una arquitectura de datos completa que integra equipos especializados, las 5 V del Big Data y soluciones tecnológicas modernas como MongoDB y la Arquitectura Lambda sobre un Data Lake.

## Conceptos clave

- Ciencia de Datos y sus raíces históricas: Florence Nightingale como primera científica de datos; el Diagrama de Venn de Drew Conway (estadística + programación + dominio del problema).
- Big Data – Las 5 V: Volumen, Velocidad, Variedad, Veracidad y Valor, aplicadas al contexto de e-commerce.
- Herramientas modernas de Big Data: Apache Spark (procesamiento distribuido), Apache Kafka (streaming en tiempo real) y Snowflake (soluciones en la nube con escalabilidad).
- Algoritmos básicos: Factorial, búsqueda lineal y ordenamiento burbuja expresados en pseudocódigo.
- Notación científica y escalas de datos: De megabytes (10⁶) a terabytes (10¹²).
- Bases de datos NoSQL – MongoDB: Tipo documental, esquema flexible, formato BSON/JSON nativo, escalabilidad horizontal con sharding.
- Arquitectura Lambda + Data Lake: Capa Batch para procesamiento histórico y capa Speed para procesamiento en tiempo real.
- Perfiles del equipo de datos: Data Engineer, Data Scientist, Data Analyst, Data Governance Analyst y MLOps Engineer.
